# Escape Sequence in String

| Escape Sequence  | Description     |
| ---------------- | --------------- |
| \n               | New Line        |
| \t               | Tab             |
| \r               | Carriage Return |
| \b               | Backspace       |
| \f               | Form Feed       |
| \'               | Single Quote    |
| \"               | Double Quote    |
| \\               | Backslash       |
| \u               | Unicode         |
| \x               | Hexadecimal     |
| \0               | Null            |
| \v               | Vertical Tab    |
| \ooo             | Octal           |
| \xhh             | Hexadecimal     |
| \u{hhhh}         | Unicode         |
| \u{hhhhhhhh}     | Unicode         |
| \u{hhhhhhhhhhhh} | Unicode         |
